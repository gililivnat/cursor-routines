# Week Planning

Run on **Monday** to set priorities and focus areas for the week.

---

## Skill

**Skill:** `skills/plan-week/SKILL.md`
**State field:** `weekly.plan_week`

## What It Does

Reviews inputs to help set his weekly focus:
- Outstanding tasks and carry-overs
- Scheduled tasks due this week (from `your scheduled tasks file`)
- Key meetings coming up
- Last week's summary (if available) for context and continuity

Produces a short, prioritised focus list for the week.

---

## Completion

When done, set `weekly.plan_week` to `true` in `state.md`.
